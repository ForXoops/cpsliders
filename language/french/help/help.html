Help: Complus Sliders

Description

Le module Complus Sliders permet d'afficher et de personnaliser des carrousels (images affichées à la suite, comme un diaporama)

Les Sliders

Un slider est un composant qui regroupe un ou plusieurs éléments (voir ci-dessous). Celui-ci permet de contrôler ce qui sera affiché.
Celui-ci contient:

  • Un titre, afin de l'identifier


Les Éléments

Un élément permet de renseigner toutes les informations que le slider lié à besoin pour l'afficher.
Il contient les informations suivantes:

  • Un titre
  • Une description
  • Un lien
  • Un ordre
  • Une image
  • Le nom du slider lié
